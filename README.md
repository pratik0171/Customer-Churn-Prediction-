#  Customer Churn Prediction System

This project is a Machine Learning-based system to predict whether a customer is likely to churn (leave the service) or not.

The model is trained on telecom customer data and deployed using Streamlit.



##  Features

- Data Cleaning & Preprocessing
- Handling Imbalanced Data
- Logistic Regression Model
- Model Evaluation
- Web Application using Streamlit
- Real-Time Prediction



##  Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle
